# CoinSight 插件

## 简介
一款获取加密货币行情信息的chatgpt-on-wechat插件，提供实时加密货币市场数据查询功能。

## 安装
确保安装了Python环境并通过pip安装以下依赖：
pip install requests pandas

## 功能列表
输入“币圈行情”获取可用查询指令列表，包括：
- 山寨指数
- 彩虹价格表
- 贪婪恐慌指数
- BTC/ETH统治力指数
- ahr999定投指数

输入对应数字即可获取内容
